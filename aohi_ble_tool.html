Protocol reference (from firmware RE)

BLE frame: 55 AA · dir (01=app→device, 00=device→app) · idx · cmd · 00 · len(2B) · payload · chk
Length: TX big-endian, RX little-endian · chk = Σ&0xFF · write→07af27a5(0x22), notify←0x1234(0x24)

01 GetDeviceInfo 04 WiFi scan → SSID list 05 ConnectWiFi (SSID[33]+PW[33]) 09 Get WiFi config 0C SaveID 10 SetCloudHost (3×host)
device→app 03 WiFi status: 1=fail · 2=OK · 3=cloud-reg · 4=ready

App command classes: AcceptBaseInfo / AcceptDynamicInfo / AcceptSchedule / AcceptState / ConnectWifi / ControlDevice / DeviceFirmwareUpdate · Cloud: wss://iotservice.iaohi.com:443/ws/iot1/
